# ClearCut — Write objectives that actually mean something.

> Build, grade and export SMART performance objectives — without the guesswork.

**[→ Try it live](https://tadisweb.github.io/ClearCut/)**

---

## What is this?

Most performance frameworks tell you to write SMART objectives but give you no real help doing it. ClearCut is the writing environment that fills that gap — a structured builder with real-time SMART checking, a visual framework guide, and one-click export, all in a single file that runs entirely in your browser.

No sign-up. No data sent anywhere. Nothing installed.

---

## Features

**Welcome & Guidance**
- Pay for Performance rationale explained upfront
- Step-by-step writing guidance with a clear SMART breakdown
- Common mistakes to avoid before you start

**Framework View**
- Visual breakdown of the 70/30 performance split — The What and The How
- The four behavioural dimensions explained in context

**Build Objectives (The What)**
- Structured inputs across Financial, Above-Brand, and People & Engagement categories
- Real-time SMART checker flags issues as you type

**The How — Behavioural Objectives**
- Dedicated inputs for all four behavioural dimensions: Lead, Shape, Connect, Deliver
- SMART checker adapted for behavioural language

**Review & Export**
- Full review of all objectives in one place
- Copy to clipboard for pasting into any system
- CSV export (UTF-8 BOM encoded for clean Excel compatibility)
- CSV import to reload a previous session or pick up where you left off

---

## How to use

1. Open the live link above in any modern browser
2. Start in the **Welcome** tab to understand the framework and common pitfalls
3. Move to **Build Objectives** and write your objectives for each category
4. Complete your behavioural objectives in **The How**
5. Head to **Review & Export** to check everything, then export as CSV or copy to clipboard

---

## Tech stack

A single HTML file — React 18 and Babel loaded from CDN, no build step, no dependencies, no backend. The only external resource is the [Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans) font loaded from Google Fonts.

---

## Running locally

No setup needed. Just download `index.html` and open it in a browser.

```
open index.html
```

---

## Privacy

ClearCut runs entirely client-side. Nothing you write ever leaves your browser. No analytics, no tracking, no accounts.
